## How to capture

1. Start the app: `npm run electron:dev`
2. Use the demo scan data
3. Windows: Win+Shift+S (Snip & Sketch)
4. macOS: Cmd+Shift+4 (region capture)
5. Save at native resolution (2x on Retina displays)
